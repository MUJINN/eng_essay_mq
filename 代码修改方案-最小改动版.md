#### 完整替换后的函数

```python
def _parser_one_essay(self, subjectId, blockId, taskKey, grade, totalScore, students):
    """
    使用大模型批改英语作文 (替换原来的GEC+AES双引擎)
    :param subjectId: 科目ID
    :param blockId: 版块ID
    :param taskKey: 任务key
    :param grade: 年级
    :param totalScore: 总分
    :param students: 学生列表
    :return: 批改结果
    """
    students_res = list()  # "students" filed call interface result

    for student in students:
        uploadKey = student['uploadKey']
        post_text = student['post_text']

        # 移除换行符
        if '\n' in post_text:
            post_text = re.sub("\n", "", post_text)

        # ⭐ 调用大模型API (替换原来的GEC+AES)
        log_server.logging('>>> Do LLM Essay Correction ...')
        try:
            correction_result = self._correct_essay(
                grade=grade,
                totalScore=totalScore,
                content=post_text,
                model='llm',  # ⭐ 使用大模型
                task='en-app',
                problem=None  # AYX英语没有problem参数
            )

            if correction_result is not None:
                # ⭐ 大模型返回的数据已经包含所需字段
                # 并且score已经转换为0-1比例
                every_student_info = {
                    'uploadKey': uploadKey,
                    'score': correction_result.get('score', 0),  # 已转换为0-1
                    'highlight': correction_result.get('highlight', []),
                    'error_set': correction_result.get('error_set', [])
                }
                log_server.logging(f"LLM Correction Result: uploadKey={uploadKey}, score={every_student_info['score']}")
            else:
                # API调用失败,返回空结果
                every_student_info = {
                    'uploadKey': uploadKey,
                    'score': 0,
                    'highlight': [],
                    'error_set': []
                }
                log_server.logging(f"LLM Correction Failed: uploadKey={uploadKey}")

        except Exception as e:
            log_server.logging('LLM Correction ERROR: {}'.format(e))
            every_student_info = {
                'uploadKey': uploadKey,
                'score': 0,
                'highlight': [],
                'error_set': []
            }

        students_res.append(every_student_info)

    # 组装最终结果
    correction_res = {
        'subjectId': subjectId,
        'blockId': blockId,
        'taskKey': taskKey,
        'students': students_res
    }

    log_server.logging("exit _parser_one_essay")
    return correction_res
```

## ⚠️ 注意事项

### 1. API调用参数

- ✅ `model='llm'` - 指定使用大模型
- ✅ `task='en-app'` - 任务类型
- ✅ `totalScore` - 总分(必须传递)
- ✅ `grade` - 年级
- ✅ `content` - 作文内容(已去除换行符)
- ❌ `problem=None` - AYX英语没有题目要求

### 2. 分数转换

- 大模型返回: `{"score": 13}` (真实得分)
- 转换后: `{"score": 0.8667}` (13/15)
- 前端显示: `0.8667 * 15 = 13分`

### 3. 错误处理

- API调用失败 → 返回 `score=0, highlight=[], error_set=[]`
- 异常捕获 → 记录日志,返回空结果
- 不影响其他学生的批改

### 4. 日志监控

关键日志:
```

```

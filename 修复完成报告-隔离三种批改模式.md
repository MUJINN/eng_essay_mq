### 修改2: 根据model选择API地址 (第118-131行)

**修改前**:
```python
def _correct_essay(self, grade, totalScore, content, model='ayx', task='en', problem=None):
    log_server.logging("enter _correct_essay")

    params = {...}

    try:
        # 直接使用self.correct_essay_api

```

**修改后**:
```python
def _correct_essay(self, grade, totalScore, content, model='ayx', task='en', problem=None):
    log_server.logging("enter _correct_essay")

    params = {...}

    # 根据model选择API地址
    # model='llm' → 使用大模型API (N8N)
    # model='ayx'/'dianxin' → 使用原API (Dify/电信)
    if model == 'llm':
        api_url = self.llm_essay_api
    else:
        api_url = self.correct_essay_api

    try:
        # 使用选择后的api_url
        for i in range(3):
            response = post_request(api_url, json=params)
```

**说明**:
- `model='llm'` → 使用 `llm_essay_api` (N8N)
- `model='ayx'/'dianxin'` → 使用 `correct_essay_api` (原API)

### 修改3: 条件分数转换 (已保留,第140-143行)

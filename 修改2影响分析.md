## 🔧 解决方案

### 方案A: 条件判断 (推荐) ⭐

**修改第135-139行的分数转换逻辑**:

```python
# 只对大模型进行分数转换
if model == 'llm' and correct_result and 'score' in correct_result:
    real_score = correct_result.get('score', 0)
    score_ratio = real_score / totalScore if totalScore > 0 else 0
    correct_result['score'] = score_ratio
```

**优点**:
- ✅ 只对 `model='llm'` 进行分数转换
- ✅ 语文Dify和电信不受影响
- ✅ 安全且向后兼容

```python
def _correct_essay(self, grade, totalScore, content, model='ayx', task='en', problem=None):
    log_server.logging("enter _correct_essay")

    # 根据model选择API地址
    if model == 'llm':
        api_url = self.llm_essay_api
    else:
        api_url = self.correct_essay_api

    params = {
        'task': task,
        'model': model,
        'content': content,
        "problem": problem,
        "grade": grade,
        "totalScore": totalScore
    }

    # ... 后续代码
    response = post_request(api_url, json=params)
```

**优点**:
- ✅ 完全隔离,不互相影响
- ✅ 易于维护和调试
- ✅ 可以独立切换每个API

# 兼容性分析 - 四种作文批改模式

## 📋 当前系统支持的批改模式

### 1️⃣ 自研AYX模型 - 英语 (`model='ayx', task='en-app'`)

**调用路径**:
```
parser_one_essay()
  ↓
if model == 'ayx' and task == 'en-app':
  ↓
_parser_one_essay()  # 原来的自研模型
  ↓
调用 GEC + AES 两个引擎
```

**返回格式**:
```python
{
    'score': 0.8667,  # 0-1比例
    'highlight': [...],
    'error_set': [...]
}
```

---

### 2️⃣ 自研AYX模型 - 语文 (`model='ayx', task='ch-app'`)

**调用路径**:
```
parser_one_essay()
  ↓
else:  # 不满足上面的if条件
  ↓
# 进入基于大模型的批改接口
  ↓
if model == 'ayx':
  ↓
_correct_essay_by_dify()  # 调用Dify接口
```

**返回格式**:
```python
{
    'percentageScore': 86.67,  # 百分比
    'highlight': [...],
    'error_set': [...]
}

# 转换为:
{
    'score': 0.8667,  # percentageScore / 100
    'highlight': [...],
    'error_set': [...]
}
```

**转换代码** (第510-514行):
```python
percentageScore = correction_result['percentageScore']
if isinstance(percentageScore, str):
    every_student_info['score'] = 0
else:
    every_student_info['score'] = correction_result['percentageScore'] / 100.0
```

---

### 3️⃣ 电信模型 (`model='dianxin'`)

**调用路径**:
```
parser_one_essay()
  ↓
elif model == 'dianxin':
  ↓
_correct_essay_by_dianxin()
```

**返回格式**: 与AYX语文相同,包含 `percentageScore`

### 4️⃣ 大模型 (`model='llm'`) ⭐ 新增

### 修改2: 在 `parser_one_essay()` 中添加大模型分数转换

**文件**: `english_write.py`
**位置**: 第508-525行

**之前**:
```python
if correction_result is not None and isinstance(correction_result, dict):
    # 阅卷那边的score其实是百分比
    percentageScore = correction_result['percentageScore']
    if isinstance(percentageScore, str):
        every_student_info['score'] = 0
    else:
        every_student_info['score'] = correction_result['percentageScore'] / 100.0
    every_student_info['highlight'] = correction_result.get('highlight', [])
    # ... 其他字段
```

**之后**:
```python
if correction_result is not None and isinstance(correction_result, dict):
    # ⭐ 根据model类型处理分数
    if model == 'llm':
        # 大模型返回真实得分,需要转换为0-1比例
        real_score = correction_result.get('score', 0)
        every_student_info['score'] = real_score / totalScore if totalScore > 0 else 0
    else:
        # AYX语文/电信返回percentageScore,需要除以100
        percentageScore = correction_result.get('percentageScore', 0)
        if isinstance(percentageScore, str):
            every_student_info['score'] = 0
        else:
            every_student_info['score'] = percentageScore / 100.0

    every_student_info['highlight'] = correction_result.get('highlight', [])
    # ... 其他字段保持不变
```

**位置**: 第131-135行

response_data = response.json()

## 一、完整输入格式

### 🔷 HTTP 请求格式

```http
POST /correct_essay HTTP/1.1
Content-Type: application/json
Authorization: Bearer your-api-key

{
  "task": "en-app",
  "model": "llm",
  "content": "I'm Li Hua. I am writing to apply for the position...",
  "problem": "假定你是李华，写一封英文申请信",
  "grade": "高二",
  "totalScore": 15
}
```

### 🔷 JSON Schema

```json
{
  "type": "object",
  "required": ["task", "model", "content", "grade", "totalScore"],

    "task": {
      "type": "string",

      "type": "string",

    }
  }
```

### 🔷 可选字段

#### 2.5 totalScore（总分）⭐ 新增必需

```json
"totalScore": 15
```

| 属性 | 说明 |
|------|------|
| **类型** | `int` |
| **必需** | ✅ 是 |
| **可选值** | 15, 25, 60等 |
| **说明** | 作文总分 |
| **用途** | 大模型根据满分输出真实得分（如11.3/15分） |

**为什么必需？**
- ✅ 大模型需要知道满分标准
- ✅ 大模型直接输出真实得分
- ✅ 后端不需要再做转换

**示例**：
- 应用文：15分
- 读后续写：25分
- 语文作文：60分

#### 6.4 大模型批改函数（新增）

**文件**：`english_write.py`
**位置**：建议在 `_correct_essay_by_dianxin()` 函数后添加

```python
def _correct_essay_by_llm(self, grade, totalScore, content, task='en-app', problem=None):
    """
    使用大模型批改英语作文

    Args:
        grade: 年级
        totalScore: 总分
        content: 学生作文内容
        task: 任务类型
        problem: 作文题目

    Returns:
        批改结果字典（与AYX格式一致）
    """
    log_server.logging("enter _correct_essay_by_llm")

    # 构建请求参数
    params = {
        'task': task,
        'model': 'llm',
        'content': content,
        'problem': problem,
        'grade': grade
    }

    try:
        # 调用大模型API
        for i in range(3):
            response = post_request(self.correct_essay_api, json=params)
            if response is not None:
                break
            else:
                time.sleep(0.1)

        # 解析响应
        response_data = response.json()
        llm_result = response_data.get('data', None)

        # 数据转换（大模型输出 → AYX格式）
        if llm_result is not None and isinstance(llm_result, dict):
            log_server.logging("llm result transform")

            # 初始化结果字典
            correction_result = {
                'highlight': [],
                'error_set': [],

                # 其他字段保持默认值
                'comment': '',
                'suggestion': '',
                'improveArticle': '',
                'topicScore': 0,
                'contentScore': 0,
                'structureScore': 0,
                'languageScore': 0,
                'errorCorrection': [],
                'expression': []
            }

            # 转换百分比分数
            if 'percentageScore' in llm_result:
                percentageScore = llm_result['percentageScore']
                correction_result['percentageScore'] = percentageScore
                correction_result['totalScore'] = percentageScore

            # 复制highlight和error_set
            if 'highlight' in llm_result:
                correction_result['highlight'] = llm_result['highlight']
            if 'error_set' in llm_result:
                correction_result['error_set'] = llm_result['error_set']

            log_server.logging("exit _correct_essay_by_llm")
            return correction_result
        else:
            log_server.logging("llm result is None or invalid")
            return None

    except Exception as e:
        log_server.logging("Exception in _correct_essay_by_llm: {}".format(str(e)))
        return None
```
